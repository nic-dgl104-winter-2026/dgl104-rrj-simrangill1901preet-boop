## What I Learned

This week helped me understand how OOP is used to organize programs in a structured and realistic way. Instead of focusing only on functions, OOP allows developers to model real-world entities using objects that contain both data and behavior.

I learned that OOP helps reduce complexity by dividing large programs into smaller, manageable components. Each class is responsible for its own data and functionality, which makes the overall system easier to maintain.

Concepts like inheritance and polymorphism also stood out to me because they allow flexibility and code reuse. This means developers can build on existing structures instead of starting from scratch every time.

I also learned that Python is a multi-paradigm language, meaning it supports OOP along with other approaches. This gives developers more flexibility when designing programs.
